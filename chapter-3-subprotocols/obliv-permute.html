Oblivious random permutation

Oblivious random permutation

The procedure

This is the main building block of the oblivious sort. We briefly describe here the recent construction of Peeter Laud; it specifically requires SPDZ, so it might need to be replaced in case one wants to use different sharing scheme.

For each pair of parties i,ji, ji,j, we need to precompute the following data Xi,XjX_i, X_jXi​,Xj​ for each party:

  1. The party iii picks a permutation π\piπ. Honest parties are assumed to pick it uniformly at random.
  2. Pick a pair of random vectors xˉ,yˉ∈RFm\bar x, \bar y \in_R \mathbb{F}^mxˉ,yˉ​∈R​Fm. Set zˉ=π(xˉ)−yˉ\bar z = \pi(\bar x) - \bar yzˉ=π(xˉ)−yˉ​.
  3. Set Xi=(π,zˉ)X_i = (\pi, \bar z)Xi​=(π,zˉ), Xj=(xˉ,yˉ)X_j = (\bar x, \bar y)Xj​=(xˉ,yˉ​).

In practice, this can be done in the offline phase in a few different ways.

Then, this precomputed data can be used to construct from a vector aˉj=(a1,...,am)j\bar a_j = (a_1, ..., a_m)_jaˉj​=(a1​,...,am​)j​ belonging to the jjj-th party a permuted vector [π(aˉ)]ij=([aπ(1)],...,[aπ(m)])ij[\pi(\bar a)]_{ij} = ([a_{\pi(1)}], ..., [a_{\pi(m)}])_{ij}[π(aˉ)]ij​=([aπ(1)​],...,[aπ(m)​])ij​ shared between iii and jjj in a following way:

  1. jjj sends to iii the vector aˉ′=aˉ+xˉ\bar a' = \bar a + \bar xaˉ′=aˉ+xˉ
  2. jjj sets π(aˉ)j=−yˉ\pi (\bar a)_j = -\bar yπ(aˉ)j​=−yˉ​
  3. iii sets π(aˉ)i=π(aˉ′)−zˉ\pi(\bar a)_i = \pi(\bar a') - \bar zπ(aˉ)i​=π(aˉ′)−zˉ

Now, the oblivious random permutation protocol of a shared value [aˉ][\bar a][aˉ] is the following:

Repeat for each party iii: Permute with πi\pi_iπi​ each of the shares aˉj\bar a_jaˉj​, setting the new shares value aˉj:=−yˉij\bar a_j := -\bar y_{ij}aˉj​:=−yˉ​ij​; set iii-th share to be the sum of discrepancies: aˉi:=∑j=1;j≠in[aˉij′]\bar a_i :=\overset{n}{\underset{j = 1; j \neq i}\sum}[\bar a'_{ij}]aˉi​:=j=1;j≠i∑​n​[aˉij′​].

This requires nnn rounds and total O(n2m)O(n^2 m)O(n2m) communication.

Active security

The protocol above is passively secure. The active check can be made post-factum: to check that a pair of vectors aˉ,bˉ\bar a, \bar baˉ,bˉ are permutations of each other, we sample a random value ttt and check ∑1ai−t=∑1bi−t\sum \frac{1}{a_i - t} = \sum \frac{1}{b_i - t}∑ai​−t1​=∑bi​−t1​.

Offline phase

Offline phase can be done in various ways. We are currently planning to use butterfly network to apply a permutation π\piπ to a shared value [x][x][x] in the offline phase, but other approaches are also possible (mixnet approach might outperform butterfly network for large mmm), this is largely TBD.
